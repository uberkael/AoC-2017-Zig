This project features my naive solutions for the [Advent of Code 2017](https://adventofcode.com/2017) in Zig.
They are simple implementations designed to practice Rust programming.

## Project Structure

- `src/`: Source code.
- `inputs/`: Inputs for the problems.
Place your inputs in `inputs/n/input`, where `n` is the day number.

## Usage

Clone this repository and run:

```bash
zig build run -Doptimize=ReleaseFast
# or
zig build run
```

Or run the tests

```bash
zig build test --summary all
```
